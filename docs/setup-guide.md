# 1. Objective

To create a centralized documentation website for storing technical notes and guides related to:

- Commvault
- DevOps
- Kubernetes
- Docker
- CI/CD
- Linux
- GitHub Actions

The website will be hosted using GitHub Pages and built using Docusaurus.

---

# 2. What is Docusaurus?

Docusaurus is a static site generator developed by Meta (Facebook) that converts Markdown files into a professional documentation website.

## Benefits

- Easy to maintain
- Markdown-based
- Automatic sidebar generation
- Search support
- Dark mode
- GitHub Pages deployment
- Version control integration

# 7. sidebars.js

## Purpose

Controls the navigation menu.

### File

```javascript
const sidebars = {
  tutorialSidebar: [
    {
      type: 'autogenerated',
      dirName: '.',
    },
  ],
};

module.exports = sidebars;
```

### Why?

Automatically generates sidebar based on folder structure.

---

# 8. docusaurus.config.ts

## Purpose

Main configuration file.

### Website URL

```ts
url: 'https://deeptiyadavsd1111.github.io',
baseUrl: '/DY_Documents/',
```

### Home Page

```ts
docs: {
  routeBasePath: '/',
  sidebarPath: './sidebars.js',
},
```

### Meaning

```text
https://deeptiyadavsd1111.github.io/DY_Documents/
```

Opens documentation directly.

### Broken Links

```ts
onBrokenLinks: 'throw',

markdown: {
  hooks: {
    onBrokenMarkdownLinks: 'warn',
  },
},
```
